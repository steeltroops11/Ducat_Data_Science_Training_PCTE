# Solar Power Generation Prediction

## Project Overview

This project predicts **AC Power Generation** of a solar power plant using environmental and weather-related parameters.

The model is trained using historical solar plant data and uses **Linear Regression** to estimate power generation based on real-time conditions.

---

## Problem Statement

Solar power generation depends on several environmental factors such as temperature and solar irradiation.

Accurate prediction of power generation can help:

* Monitor plant performance
* Improve energy planning
* Detect abnormal generation patterns
* Support efficient power management

---

## Features Used

The model uses the following input features:

| Feature             | Description                           |
| ------------------- | ------------------------------------- |
| Ambient Temperature | Surrounding environmental temperature |
| Module Temperature  | Temperature of solar panels           |
| Irradiation         | Amount of solar radiation received    |

---

## Target Variable

| Target   |
| -------- |
| AC_POWER |

The model predicts the AC power generated by the solar plant.
